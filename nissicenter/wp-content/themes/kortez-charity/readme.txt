== Description ==

Kortez Charity is a clean and minimal charity theme especially for Charity, NGO, Non-Profit Organization or Foundation, Campaign & Event Donation, Religion or a Fundraising website. This theme supports elementor and have lots of features.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
